# SEO 최적화
- `<head>` 사이에 코드를 넣어주세요.

```
<title>웹사이트 제목</title>
<meta name="description" content="웹사이트 설명"/>
<meta name="keywords" content="웹사이트 키워드 예시) a, b, c, d, e, f, g"/>
<meta name="author" content="소유자 이름 (?)"/>
<meta name="og:type" content="웹사이트 형식"/>
<meta name="og:title" content="임베드에 표시할 제목"/>
<meta name="og:site_name" content="임베드에 표시할 웹사이트 이름"/>
<meta name="og:description" content="임베드에 표시할 웹사이트 설명"/>
<meta name="og:url" content="임베드에 표시할 URL (제목URL)"/>
<meta name="og:image" content="임베드에 표시할 웹사이트 사진 URL을 작성하세요."/>
```
